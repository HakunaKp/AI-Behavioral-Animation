Description:
The enemy AI randomly searches for the player. 
If the player is in enemy's field of view, enemy will increase speed and chase player.
To win, the player must get to the coins (spacebar to go through wall).
But the player loses if it is caught by the enemy AI.

User Input Controls:
- AWSD/Arrow Key player movement
- Mouse movement controls player direction
- Player shoots while mouse is clicked
- Space bar toggles wall collisions to reach coins
- Slider adjust player's movement speed

Game Features:
- User has control of the fixed wall's collisions.
- Visualizations for player and enemy FOV's
- Goal position is coins. Win when player's FOV finds coins.
- Parameterized player's speed, adjust with slider
